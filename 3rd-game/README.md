# 小朋友下樓梯 (Terminal Edition)

一個以 Python 3 + curses 製作的終端機版「小朋友下樓梯」遊戲。

## 執行方式

```bash
cd 3rd-game
python main.py
```

> **Windows 使用者**請先安裝 curses 支援：
> ```bash
> pip install windows-curses
> ```

## 遊玩方式

| 按鍵 | 功能 |
|------|------|
| `a` / `←` | 向左移動 |
| `d` / `→` | 向右移動 |
| `p`       | 暫停 / 繼續 |
| `q`       | 離開 |
| `r`       | 重新開始（Game Over 畫面） |

## 平台說明

| 符號 | 類型 | 效果 |
|------|------|------|
| `[====]` | 普通平台 | 可正常站立 |
| `<####>` | 傷害平台 | 踩到扣 1 HP（附火花特效） |
| `(++++)` | 補血平台 | 踩到回 1 HP（附光點特效） |
| `{~~~~}` | 崩壞平台 | 踩後閃爍再消失（附碎片特效） |
| `/^^^^\ ` | 彈簧平台 | 踩到向上彈射（附彈射特效） |
| `<---->` | 移動平台 | 左右來回移動（帶光暈） |

## 遊戲特色

- **粒子特效系統**：著陸揚塵、傷害火花、補血光點、彈簧噴射、崩壞碎片、移動尾跡
- **螢幕震動**：受傷與彈跳時的畫面震動回饋
- **螢幕閃光**：受傷紅閃、補血綠閃
- **連擊系統**：連續踩平台累積 combo，加成分數，高 combo 頭頂發光
- **動態色彩**：彩虹標題、等級越高邊框越炫、高分彩虹特效
- **多層視差星空**：三層不同速度的背景星空
- **速度線**：高速時側邊出現速度線
- **危機警告**：靠近頂部時閃爍警告 + 側邊驚嘆號
- **玩家動畫**：面朝方向切換、下落動畫、無敵閃爍
- **難度漸進**：每 30 秒提升等級，捲動速度緩慢加快
- **分數排行榜**：TOP 5 紀錄自動存檔

## 遊戲規則

- 角色受重力持續下落，必須踩到平台才能存活
- 畫面持續向上捲動（約每 5 秒一格），製造「不斷下樓」的壓迫感
- 玩家掉出畫面或 HP 歸零即 Game Over
- 分數隨存活時間累積，連擊可獲得額外加成
- 難度每隔約 30 秒提升一次
- Game Over 後分數存入排行榜（TOP 5）

## 檔案結構

```
3rd-game/
├── main.py           程式入口
├── game.py           主遊戲迴圈 & 狀態機 & 粒子整合
├── entities.py       玩家、平台、粒子資料結構
├── renderer.py       curses 渲染（含所有視覺特效）
├── input_handler.py  非阻塞輸入處理
├── constants.py      全域常數
├── scores.json       排行榜紀錄（自動生成）
└── README.md         本文件
```

## 已知限制

- 需要終端機至少 **52 × 29** 字元大小（建議更大以顯示側邊圖例）
- curses 在 Windows 原生終端上可能有顯示問題，建議使用 WSL / Git Bash
- 部分 Unicode 方框字元在某些終端機可能無法正確顯示
