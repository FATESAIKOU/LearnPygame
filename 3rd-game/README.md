# 小朋友下樓梯 (Terminal Edition)

一個以 Python 3 + curses 製作的終端機版「小朋友下樓梯」遊戲。

## 執行方式

```bash
python main.py
```

> **Windows 使用者**請先安裝 curses 支援：
> ```bash
> pip install windows-curses
> ```

## 遊玩方式

| 按鍵 | 功能 |
|------|------|
| `a` / `←` | 向左移動 |
| `d` / `→` | 向右移動 |
| `p`       | 暫停 / 繼續 |
| `q`       | 離開 |
| `r`       | 重新開始（Game Over 畫面） |

## 平台說明

| 符號 | 類型 | 效果 |
|------|------|------|
| `====` | 普通平台 | 可正常站立 |
| `XXXX` | 傷害平台 | 踩到扣 1 HP |
| `++++` | 補血平台 | 踩到回 1 HP |
| `~~~~` | 崩壞平台 | 踩一下消失 |
| `^^^^` | 彈簧平台 | 踩到向上彈射 |
| `----` | 移動平台 | 左右來回移動 |

## 遊戲規則

- 角色受重力持續下落，必須踩到平台才能存活
- 畫面持續向上捲動，製造「不斷下樓」的壓迫感
- 玩家掉出畫面或 HP 歸零即 Game Over
- 分數隨存活時間累積，難度每隔一段時間提升
- Game Over 後分數存入排行榜（TOP 5）

## 檔案結構

```
3rd-game/
├── main.py         程式入口
├── game.py         主遊戲迴圈 & 狀態機
├── entities.py     玩家、平台資料結構
├── renderer.py     curses 渲染
├── input_handler.py  非阻塞輸入處理
├── constants.py    全域常數
└── scores.json     排行榜紀錄（自動生成）
```

## 已知限制

- 需要終端機至少 **52 × 26** 字元大小
- curses 在 Windows 原生終端上可能有顯示問題，建議使用 WSL / Git Bash
- 中文字元 `♥♡` 在部分終端機可能無法顯示，可改為 `*` / `-`
