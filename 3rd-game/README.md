# 小朋友下樓梯 — NS-SHAFT Terminal Edition 🎮

一款在終端機中遊玩的「小朋友下樓梯」遊戲，使用 Python 3 + curses 製作。

## 執行方式

```bash
cd 3rd-game
python3 main.py
```

### 系統需求

- Python 3.6+
- 終端機支援 Unicode 字元
- 終端機大小至少 66×35
- Linux / macOS（原生 curses 支援）
- Windows 使用者請安裝：`pip install windows-curses`

## 遊玩方式

### 操作

| 按鍵 | 動作 |
|------|------|
| `A` / `←` | 左移 |
| `D` / `→` | 右移 |
| `P` | 暫停 |
| `Q` / `ESC` | 退出 |
| `Enter` / `Space` | 確認（選單中） |

### 遊戲規則

- 玩家角色 `☺` 會持續因重力往下掉
- 必須踩在平台上來生存
- 畫面會持續向上捲動，產生往下走的壓迫感
- 碰到天花板的尖刺會扣血
- 掉出畫面底部或生命歸零則遊戲結束

### 平台種類

| 字元 | 類型 | 效果 |
|------|------|------|
| `═══` | 普通平台 | 安全站立 |
| `▓▓▓` | 傷害平台（紅色） | 踩到扣血 |
| `♥♥♥` | 補血平台（綠色） | 踩到回血 |
| `~~~` | 移動平台（青色） | 會左右移動 |
| `░░░` | 崩壞平台（紫色） | 踩上後會崩壞消失 |
| `▲▲▲` | 彈簧平台（黃色） | 踩到會彈跳 |

### 分數系統

- 每次畫面捲動 +10 分
- 每存活 1 秒 +1 分
- 遊戲結束後顯示最終分數
- 支援單次遊戲最高分紀錄

### 難度系統

- 每 15 秒難度提升
- 捲動速度逐漸加快
- 傷害 / 崩壞平台出現機率增加
- 普通平台機率降低

## 專案結構

```
3rd-game/
├── main.py           # 程式入口
├── game.py           # 主遊戲迴圈、物理、碰撞
├── entities.py       # 玩家、平台、粒子特效
├── renderer.py       # 終端機渲染（curses）
├── input_handler.py  # 輸入控制
├── constants.py      # 常數設定
└── README.md         # 本文件
```

## 已知限制

- curses 在 Windows 需額外安裝 `windows-curses`
- 終端機必須支援 Unicode（UTF-8）
- 不同終端機對 Unicode 字元寬度的處理可能不同
- 無音效支援（終端機環境限制）
